Contoh Project Python 3 Dasar Koleksi 1

RAKIFSUL February 18, 2024

A. Pendahuluan

Python 3 adalah bahasa pemrograman yang banyak kegunaannya.

Salah satu ciri dari Python 3 adalah penggunaan indentation dalam penulisan script-nya.

B. Karya Apakah Ini?

Karya ini adalah contoh project Python 3 dasar koleksi 1.

Isinya terdiri dari beberapa project Python 3 sederhana yang dikumpulkan.

C. Fitur

Sebagai contoh kode, karya ini bisa menjadi contoh untuk:

  • Cara membaca csv.
  • Cara menulis csv.
  • Cara menggunakan file.
  • Cara menggunakan json.
  • Cara menggunakan modul sendiri.
  • Cara menggunakan modul os.
  • Cara menggunakan modul sys.
  • Cara menggunakan modul _thread.
  • Cara menggunakan modul threading.
  • Cara menggunakan modul urllib.

D. Peringatan

BACA README-nya terlebih dahulu sebelum mencoba, jika ada.

E. Disclaimer

  • Source code hanya teruji di Windows 11 dengan Python 3.10.11.
