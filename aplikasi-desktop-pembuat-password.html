Aplikasi Desktop Pembuat Password

RAKIFSUL November 6, 2023 Aplikasi Desktop Python

Disclaimer: Gambar hanya ilustrasi. Hasilnya bisa jadi berbeda.

Aplikasi ini adalah aplikasi desktop yang bisa digunakan untuk membuat password.

Masalah yang Diselesaikan

Aplikasi ini berguna saat Anda membuat sebuah akun di web, yang memerlukan password yang kuat, tapi tanpa harus memikirkannya sendiri.

Fitur Aplikasi Ini

Fitur:

  • Membuat password
  • Kombinasi huruf kecil
  • Kombinasi huruf besar
  • Kombinasi angka
  • Kombinasi simbol
  • Bisa dibuat unik, jika memungkinkan
  • Bisa kopi ke clipboard

Keunggulan Aplikasi Ini

Kelebihan aplikasi ini dibandingkan yang lainnya adalah bahwa Anda bisa request fitur tambahan, tentunya dengan sejumlah biaya.

Teknologi yang Digunakan

Aplikasi ini dibuat dengan menggunakan Python3 dan wxPython.

Biaya Modifikasi

Perkiraan harga modifikasi aplikasi ini adalah:

  • Harga awal (tanpa penambahan fitur) mulai dari Rp. 5.000.000.
  • Harga bugfix mulai dari Rp. 200.000 per bugfix
  • Harga penambahan fitur mulai dari Rp. 500.000 per fitur
