Aplikasi Desktop Game Off-Road

RAKIFSUL November 5, 2023 Aplikasi Desktop Godot

Klik di Sini untuk Modifikasi

Game ini adalah game off-road sederhana dengan tema low-poly.

Deskripsi Singkat

Pemain game ini bertugas untuk menyelesaikan tiap-tiap level yang diberikan.

Saat baru dimainkan, pemain harus memilih mobil dan level yang diinginkan.

Selanjutnya, mobil ditempatkan ke dalam level yang memiliki tantangan medannya masing-masing.

Walaupun tanpa lawan, medan di level game ini cukup menantang.

Dengan sistem kontrol mobil manual yang realistis, membuat game ini lebih menantang lagi, walaupun temanya adalah low poly.

Game ini dibuat menggunakan Godot Engine.

Biaya Modifikasi

Perkiraan harga modifikasi aplikasi ini adalah:

  • Harga awal (tanpa penambahan fitur) mulai dari Rp. 20.000.000.
  • Harga bugfix mulai dari Rp. 1.000.000 per bugfix
  • Harga penambahan fitur mulai dari Rp. 2.000.000 per fitur
