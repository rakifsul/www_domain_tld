Aplikasi Desktop Game Turret

RAKIFSUL November 6, 2023 Jasa

Disclaimer: Gambar hanya ilustrasi. Hasilnya bisa jadi berbeda.

Ini adalah game turret.

Deskripsi Singkat

Jika Anda tidak tahu apa itu turret, itu adalah benda berwarna abu-abu merah yang ada di screenshot di bawah.

Game ini bertujuan untuk survive menghadapi lawan berupa robot yang di-spawn secara random dari segala arah.

Sebagai pemain, Anda harus menembaki robot-robot tersebut.

Tidak ada batasan jumlah robotnya, jadi ini adalah game survival.

Game ini dibuat dengan menggunakan Godot Engine.
