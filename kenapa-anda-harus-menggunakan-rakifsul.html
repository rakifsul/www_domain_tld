Kenapa Anda harus Menggunakan RAKIFSUL?

RAKIFSUL November 4, 2023 Blog

Selamat datang di website saya.

Sebelumnya, izinkan saya memperkenalkan diri.

Saya adalah RAKIFSUL.

Hobi saya adalah coding dan menulis.

Karena kedua hobi tersebut bisa diuangkan, maka saya memutuskan untuk membuka layanan coding.

Skill yang saya kuasai adalah HTML, CSS, JavaScript, PHP, C++, Python, dan GDScript.

Saya sudah berurusan dengan coding sejak tahun 2004.

Tahun yang cukup sulit memang, karena untuk belajar coding di masa itu, saya harus bolak-balik ke warnet untuk membaca blog-blog dan mempelajari kode yang disediakannya.

Waktu itu, skill yang pertama saya pelajari adalah C++ dan saya sangat bersyukur karena hingga saat ini C++ masih ada dan digunakan, bahkan di platform-platform besar.

Hikmah yang saya ambil dari mempelajari C++ adalah bahwa bahasa tersebut adalah bahasa yang cukup mendasar.

Akibat mempelajari bahasa pemrograman C++, saya menjadi lebih mudah untuk mempelajari bahasa pemrograman lainnya seperti JavaScript, PHP, Python dan lain-lain.

Lalu, apakah hanya bahasa pemrograman saja yang saya pikirkan?

Tentu tidak.

Software aplikasi ada yang berjalan di platform desktop, web, maupun mobile. 

Menurut pertimbangan saya, walaupun aplikasi web banyak dicari orang, resiko deployment-nya tinggi.

Itu karena jika kita salah melakukan deployment bisa jadi akan terjadi kehilangan data, kerusakan konfigurasi dan sebagainya.

Oleh karena itu, walaupun saya masih menerima project web, saya lebih cenderung ke desktop.

Menurut saya ide bagus untuk membuka pasar di platform desktop.

Apalagi, teknologi untuk pengembangan aplikasi desktop sekarang semakin maju dengan adanya Electron js dan lain-lain.

Tentunya, saya tetap menerima project web jika saya mau dan mampu.

Lalu, apa yang saya lakukan sekarang, khususnya di website ini?

Di website ini, saya mempublikasi portfolio aplikasi desktop buatan saya.

Beberapa aplikasi yang saya buat idenya masih baru, sehingga cocok bagi Anda untuk mendapatkan ide.

Jadi, apa yang saya buat itu adalah bibit software.

Akan menjadi seperti apa software tadi setelah dewasa, itu melibatkan keputusan Anda.

Lalu, apakah saya telah berhasil mengimplementasikan bibit software tersebut?

YA... Walaupun mungkin sebagian dari bagiannya saja.

Saya telah berhasil membantu client saya membuat aplikasi WhatsApp marketing bernama WBSPro.

Beberapa bagian dari aplikasi WBSPro menggunakan teknologi obfuscation yang saya terapkan di beberapa aplikasi Electron js saya.

Saya sendiri yang menemukan teknologi obfuscation tersebut.

Selain itu, dalam pemeliharaannya, saya juga membantu melakukan update yang cukup sulit.

Untungnya, saya bisa melakukannya dengan cukup cepat.

Sebelum saya mengerjakan project WBSPro, sebenarnya saya sudah pernah mengerjakan project dari luar negeri, lebih tepatnya dari Singapura melalui salah satu website freelance luar negeri.

Salah seorang client saya dari Singapura pernah meminta dibuatkan launcher untuk game simulasi pesawat terbang.

Dan lagi-lagi, saya berhasil.

Padahal untuk melakukan itu, tidak ada forum, blog, bahkan dokumentasi untuk melakukannya.

Saya benar-benar membaca kodenya dan melakukan itu.

Jika ingin merunut lebih jauh ke belakang, saya juga pernah mengerjakan project dari client yang berasal dari Turki.

Di project tersebut, saya diminta untuk menemukan algoritma baru untuk membuat sudoku number dan saya berhasil.

Bayangkan, algoritma baru untuk sudoku itu bukanlah hal yang mudah.

Nah. Sekarang Anda sudah tahu apa yang bisa saya lakukan dan apa yang sedang saya lakukan.

Sekarang pilihan Anda.

Apakah Anda ingin menggunakan saya?

Oiya, tapi disclaimer ya...

Saya tidak menjamin bahwa saya akan selalu berhasil mengerjakan project Anda.

Itu karena banyak faktor yang menentukan keberhasilan suatu project.
